# Addin Midua Photography

Static photography portfolio and client inquiry site for GitHub Pages.

## Publish With GitHub Pages

1. Push this folder to a public GitHub repository.
2. In the repository, open Settings > Pages.
3. Set Source to "Deploy from a branch".
4. Choose `main` and `/root`, then save.

The contact form uses FormSubmit and sends inquiries to `miphoto2003@gmail.com`.
The first submission may require email confirmation from FormSubmit before future form messages are delivered.
